# Project Overview - TimeMoney

> Generated: 2026-03-16 | Scan Level: Exhaustive

## Executive Summary

**TimeMoney** is a cross-platform Flutter mobile application designed to help hourly workers track their work hours and calculate payments. Users can log time entries (hours and minutes), set their hourly wage rate, and calculate total payment across all recorded entries. The app uses a local-first architecture with ObjectBox for offline data persistence and follows Clean Architecture principles with the BLoC pattern for state management.

## Architecture Classification

- **Repository Type:** Monolith
- **Architecture Pattern:** Clean Architecture with feature-based organization
- **State Management:** BLoC/Cubit pattern with Freezed events and states
- **Data Layer:** ObjectBox local database with reactive streams
- **Error Handling:** Functional (Dartz Either) with Freezed union types
- **DI Strategy:** flutter_bloc RepositoryProvider + MultiBlocProvider

## Key Features

1. **Time Entry Management** - Create, read, update, and delete work time entries (hours + minutes)
2. **Hourly Wage Configuration** - Set and update hourly pay rate (default: $15.00)
3. **Payment Calculation** - Calculate total payment based on logged hours and wage rate
4. **Reactive Data** - Real-time UI updates via ObjectBox streams
5. **Multi-platform** - Runs on iOS, Android, Web, and Windows
6. **Multi-environment** - Development, staging, and production flavors with separate databases
7. **Localization** - English and Spanish language support
